# AI-SaaS

A modern, full-stack AI-powered Software-as-a-Service application built with cutting-edge web technologies. This project demonstrates a complete implementation of a cloud-based AI platform with a responsive frontend and robust backend infrastructure.

**[Live Demo](https://ai-saas-omega-ten.vercel.app)** | **[GitHub Repository](https://github.com/XKenda/AI-SaaS)**

---

## 🎯 Overview

AI-SaaS is a comprehensive web application that integrates AI capabilities with a modern SaaS architecture. It provides users with an intuitive interface to leverage AI services while maintaining enterprise-grade security and scalability.

### Tech Stack

- **Frontend**: React 19, Vite, Tailwind CSS, Framer Motion
- **Backend**: Express.js, Node.js
- **Database**: MongoDB with Mongoose
- **AI Integration**: Google GenAI
- **Cloud Storage**: Cloudinary
- **Authentication**: JWT with bcryptjs
- **Styling**: Tailwind CSS with UI Components

---

## ✨ Features

- 🤖 **AI Integration** - Seamless integration with Google GenAI for intelligent features
- 🔐 **Secure Authentication** - JWT-based authentication with password encryption
- 🎨 **Modern UI** - Responsive design with Tailwind CSS and smooth animations using Framer Motion
- 📊 **Data Visualization** - Charts and analytics with Recharts
- 🖼️ **Image Management** - Cloud-based image storage with Cloudinary
- 📱 **Mobile Responsive** - Fully responsive design for all devices
- ⚡ **High Performance** - Optimized with Vite for fast development and production builds
- 🔄 **Real-time Updates** - Axios-based API communication
